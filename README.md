# 我的博客

## 环境配置和项目启动 📑

- **Clone：**

```text
# GitLab
git clone git@github.com:KarasYe/my-blog.git
```

- **Install：**

```text
pnpm install

# pnpm install 安装失败，推荐请升级 nodejs 到 v16.17.1+，pnpm v6.32.2+，npm v8.13.2+
```

- **Run：**

```text
pnpm docs:dev
```
